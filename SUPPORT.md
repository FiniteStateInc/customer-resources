# Support

This document provides information on getting help and support for the Finite State Customer Resources repository.

For support inquiries, please contact **support@finitestate.io**

## Getting Help

### Documentation

- Start by checking the README files in the relevant workflow stage directory
- Review the main [README.md](./README.md) for an overview
- Check examples and code comments in the shared utilities

### Issues and Questions

For questions, bug reports, or feature requests:

1. **Search existing issues** - Your question may already be answered
2. **Open a new issue** - Use the appropriate issue template if available
3. **Provide context** - Include relevant code examples, error messages, and steps to reproduce

### Community Support

- Check discussions and existing issues for community answers
- Share your use cases and examples to help others

## Reporting Problems

When reporting problems, please include:

- **Description** - What are you trying to accomplish?
- **Steps to reproduce** - Detailed steps that lead to the issue
- **Expected behavior** - What should happen?
- **Actual behavior** - What actually happens?
- **Environment** - OS, language versions, relevant dependencies
- **Code examples** - Minimal code snippets that demonstrate the issue

## Security Issues

For security-related concerns, please see [SECURITY.md](./SECURITY.md) instead of opening a public issue.

## Contributing

If you have solutions or improvements, we welcome contributions! See [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines.
