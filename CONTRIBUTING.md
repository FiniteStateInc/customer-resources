# Contributing to Finite State Customer Resources

Thank you for your interest in contributing to Finite State Customer Resources! This document provides guidelines and instructions for contributing.

## How to Contribute

### Reporting Issues

If you find a bug, have a suggestion, or want to report a problem:

1. Check if the issue has already been reported
2. Create a new issue with a clear title and description
3. Include steps to reproduce (if applicable)
4. Provide examples or code snippets when relevant

### Suggesting Enhancements

We welcome suggestions for new examples, tools, or improvements to existing content:

1. Open an issue with the `enhancement` label
2. Clearly describe the use case and benefit
3. Provide examples if possible

### Submitting Pull Requests

1. **Fork the repository** and create a branch from `main`
2. **Make your changes** following our style guidelines
3. **Test your changes** to ensure they work as expected
4. **Update documentation** if needed
5. **Submit a pull request** with a clear description

### Pull Request Guidelines

- Use clear, descriptive commit messages
- Keep pull requests focused on a single topic
- Update relevant README files if adding new content
- Ensure all examples are tested and working
- Follow the existing code style and structure

## Content Organization

When adding new content, please:

- Place it in the appropriate workflow stage directory
- Update the relevant README.md file
- Add cross-references to related content
- Follow the existing naming conventions
- Include required environment variables in documentation (see main README.md)

## Code Style

- Use clear, readable code with comments where helpful
- Include example usage in README files
- Provide error handling where appropriate
- Document any required dependencies

## Questions?

If you have questions about contributing, please:

- Check existing issues and pull requests
- Open a new issue with your question
- Contact the maintainers as specified in SUPPORT.md

Thank you for contributing to Finite State Customer Resources!
